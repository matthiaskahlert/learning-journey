Variablen mit let deklarieren

let x1 = 20; let y1 = 1; let x2 = 65; let y2 = 4; console.log(x1,y1,x2,y2);

Unterschiede zwischen const und let

const ist eine Konstante, während let sich überschreiben lässt. Beispiel let: let alter = 45 alter = 20 console.log("Alter: ", alter) Dies führt zum Ausgabewert "Alter: 20" Beispiel const: const drink = "Kaffee"; drink = "Wasser"; console.log("Drink: ", drink)

Dies führt zu einer exception da drink eine Konstante Variable ist.

index.html:59 Uncaught TypeError: Assignment to constant variable. at index.html:59:15

Mehrere Variablen deklarieren

Statt mehrere Variablen pro Zeile zu schreiben, kann man auch sogenannte Mehrfachzuweisungen nutzen.

let q,w,e; const a=10, b=20, c=30;
