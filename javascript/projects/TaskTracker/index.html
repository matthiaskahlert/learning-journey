ℹ️ Was passiert beim Speichern?

Jede neue Aufgabe wird lokal im Browser gespeichert. Damit bei Reload die Aufgaben nicht verloren gehen wird localStorage verwendet und in der Liste angezeigt. Zusätzlich wird die Aufgabe per Fetch API als POST-Request an eine Test-API (JSONPlaceholder) gesendet. Die Server-Antwort kannst du in der Browser-Konsole (F12) einsehen.
