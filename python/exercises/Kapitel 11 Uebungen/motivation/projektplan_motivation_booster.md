## 5. Ausfuehrung auf Rechnern ohne Python

Wichtig:
- Eine .py/.pyw Datei braucht lokal Python.
- Ohne Python lokal: Build als Windows .exe (z. B. mit PyInstaller).

Sinnvolle Portfolio-Variante:
- Source + Doku im Repo
- Zusaetzlich fertige .exe als Release

## 6. Audio-Lizenzen ohne Risiko

Sicherster Weg:
- CC0 oder Public Domain Audio nutzen

Wenn nicht CC0:
- Lizenz genau pruefen
- Quelle/Autor/Lizenz in README dokumentieren
- Lizenztext lokal aufheben

No-Risk-Checkliste:
- Nur Audio mit klarer Erlaubnis verwenden
- Lizenzbedingungen einhalten (z. B. Namensnennung)
- Keine ungeklaerten Dateien aus dem Netz verwenden
